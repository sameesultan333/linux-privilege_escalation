# Linux Privilege Escalation Automation Toolkit

## Overview

Automated Linux security auditing toolkit developed during internship.

## Features

- System Information Collection
- SUID Binary Discovery
- Permission Analysis
- Sudo Configuration Review
- Cron Job Enumeration
- Service Enumeration
- Linux Capability Detection
- Kernel Information Analysis

## Usage

python3 main.py

## Output

The toolkit identifies potential privilege escalation vectors and provides security recommendations.
